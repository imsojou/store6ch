## 4. 데이터 갱신 방법

매장이 신규 출점·폐점하면 앱 안에서는 고칠 수 없고 **파일을 다시 만들어야** 합니다.

1. `data/stores.json` 의 `stores` 배열을 수정 (또는 매장마스터 엑셀에서 재추출)
2. `data/앱템플릿_shell.html` 의 `__DATA__` 를 stores.json 내용으로, `__APT__` 를 apts.json 내용으로 치환
3. 결과를 `상권분석지도.html` 로 저장

`stores.json` 필드: `id`(매장ID) `ch`(채널) `nm`(매장명) `ty`(매장유형) `lt`(입점유형) `sd`/`sg`/`em`(시도/시군구/읍면동) `ad`(주소) `la`/`ln`(좌표) `tm`(지역팀) `mt`(대형팀) `c3`(3km내 경쟁매장) `nd`(최근접거리)

## 5. 주의사항

**사용자 입력값은 브라우저(localStorage)에 저장됩니다.** 다른 PC나 다른 브라우저에서 열면 보이지 않습니다. 아래 3가지는 반드시 **내보내기 탭에서 CSV로 백업**하세요.

- 매장 평수·직원수·월매출 입력값
- 직영/대리점 분류 보정
- 지역 담당팀 배정 변경

**검수 권장 항목**

- LG 매장 22개는 공식 목록에 없어 대리점으로 처리했습니다. 이 중 LG 사업장內 매장(파주사업장·사이언스파크·트윈타워)과 홈플러스 숍인숍(영주·포항·대구수성)은 실제로는 직영일 가능성이 있습니다.
- 삼성 직영 목록 중 18개(스타필드 고양, 도안, 롯데프리미엄아울렛 의왕 등)가 매장마스터에 없습니다. 신규 매장으로 추정되며 마스터 갱신이 필요합니다.
- 2028년 입주 단지는 공식 통계가 아직 없습니다(기관별 총량 전망치만 존재). 앱의 CSV 업로드로 추가할 수 있습니다.
